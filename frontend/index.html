Upload an MRI and run prediction to see the Grad-CAM++ explanation.
Why this matters (Black-box problem): A model can achieve high test accuracy by learning spurious correlations — for example, if all tumour MRIs in the training set had a bright artefact in one corner due to a faulty scanner, the model would learn to detect that artefact rather than the tumour itself. Grad-CAM++ reveals what the model actually learned by showing which pixels most strongly drove the prediction.

How it works: The gradient of the predicted class score is computed with respect to the last convolutional layer activations. Second- and third-order gradient terms (Grad-CAM++ improvement over Grad-CAM) are used to compute per-channel importance weights. The weighted activation maps are summed and upsampled to the original image resolution. Hot (red) regions = high positive influence on the prediction. Blue = low or negative influence.
